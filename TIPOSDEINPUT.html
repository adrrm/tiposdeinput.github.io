TIPOS DE INPUT

Button:

Se refiere generalmente a un botón en una interfaz gráfica que ejecuta una acción cuando se presiona.


Checkbox:

Es un elemento que permite al usuario seleccionar o deseleccionar una opción, representada por una casilla que puede estar marcada o desmarcada.


Color:

Permite que las consecuencias de caracteres de escape ANSI produzcan texto de terminal en color y que el posicionamiento del cursor funcione en MS WINDOWS.


Date:

Se usa para permitir que el usuario seleccione una fecha desde un calendario emergente.


Date-time:

Es un campo que permite a los usuarios seleccionar una fecha y una hora, combinado ambos en una sola entrada.


Email:

Es un tipo de campo de entrada en html diseñado especificamente para capturar direcciones de correo electronico. Este tipo de entrada permite que los navegadores validen automaticamente que la dirección de correo electronico tenga el formato correcto.


File:


Se usa para permitir que los usuarios seleccionen un archivo desde su dispositivo, ya sea para subirlo a un servidor o para procesarlo de alguna manera en una aplicacion web.

Hidden:

Se usa para almacenar informacion que el usuario no puede ver ni interactuar directamente, pero que puede enviarse con el formulario.


Image:

Se usa para colocar una imagen como boton para enviar un formulario. En lugar de un boton tradicional, usas una imagen que actua como el boton de envio.


Month:

Permite al usuario seleccionar un mes y un año. No se selecciona días, solo un mes y un año.


Number:

Permite que el usuario ingrese solo numeros. Ademas, puede incluir flechas de incremento y decremento.


Password:

Se usa para ocultar el texto ingresado en el campo, ideal para contraseñas. En lugar de mostrar el texto, se muestran asteriscos o puntos.


Radio:

Permite a los usuarios seleccionar una opcion de un grupo de opciones.


Range:

Se usa para crear un control deslizante donde el usuario puede seleccionar un valor dentro de un rango.


Reset:

Se usa para crear un botón que restablece todos los campos de un formulario a sus valores iniciales.


Search:

Es un campo optimizado para la búsqueda, aunque se comporta como un campo de texto.


Submit:

Se usa para crear un boton que envia el formulario.


Tel:

Es un campo de entrada diseñado para capturar numeros de telefono. No realiza validaciones especificas en los navegadores, pero puede ayudar a los usuarios a ingresar numeros de telefonos con un teclado numerico.


Text:

Es un campo estandar para ingresar texto.


Time:

Permite a los usuarios seleccionar una hora.


Url:

Permite al usuario ingresar una URL. Los navegadores validan que tengan el formato adecuado.


Week:

Permite al usuario una semana de un año.
